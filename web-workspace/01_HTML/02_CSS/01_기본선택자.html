선택자(Selector)란 무엇인가?? ⭐중요

특정 HTML요소만을 선택하려고할 때 사용하는 문법
선택자를 이용하면 원하는 요소만 골라서 Style속성을 부여할 수 있다

잘 안쓰는 방식이다

1. 전체(모든 요소)선택자


        *{
            스타일속성 : 속성값;
            스타일속성 : 속성값;
        }

        => 현재 이 HTML문서 내의 모든 요소를 다 선택해서 스타일을 부여

2. 태그 선택자

        태그명{
            스타일속성 : 속성값;
            스타일속성 : 속성값;

        }


        => 문서 내에 같은 태그
/* 3. 아이디 선택자 */ 
 /*CSS 요소들을 꼭! 식별해야 한다*/
        #아이디속성값{
            스타일 속성 : 속성값;
            스타일 속성 : 속성값;
        }

        => 문서 내의 특정 HTML요소 단 한 개 만을 선택할 때 사용
        단, 해당 요소에 반드시 id속성값이 존재해야만하며 다른 요소와 중복되면 안된다!
        id값은 반드시 하나의 문서에서 고유한 값이여야한다!!
        동일한 태그더라도 부분적으로 다르게 스타일을 부여하고 싶을 때 사용

  3. 아이디선택자
  4. 아이디선택자

4. 클래스 선택자

        .클래스속성값{
            스타일속성 : 속성값;
            스타일속성 : 속성값;
        }

        => 현재 HTML문서에서 여러 개의 요소를 선택할 때 사용
