요소의 영역 구분

margin : 요소의 바깥쪽 여백 영역 (각각 요소 서로간의 간격(여백)을 띄어주기 위함)
border :테두리 영역(경계)
padding : 내용물영역과 테두리 영역사이의거리(안쪽에서 얼만큼 여백을 줄것인지)
content : 실제 내용물이 들어가는 영역
실제요소가 차지하는 크기(영역) == content + border
