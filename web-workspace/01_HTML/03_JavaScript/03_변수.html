자바스크립트의 경우 자료형을 선언하지 않는다

자바스크립트는 총 세개의 키워드로 변수를 선언할 수 있다

* var, let, const

변수 선언 위치에 따른 전역 / 지역변수로 나눈다

* 자료형

        변수선언시에 자료형을 별도롤 선언하지 않는다
        그렇다고 자료형의 개념이 없는 것은 아니다
        =>변수에 대입되는 값에 따라서 알아서 자료형을 결정한다 : 타입추론

자료형의 종류

  • string(문자열)
  • number(숫자)
  • boolean(논리)
  • objext(객체)
  • function(함수)
  • underfined(초기화가 안된 상태/초기화를 진행되지 않는)
