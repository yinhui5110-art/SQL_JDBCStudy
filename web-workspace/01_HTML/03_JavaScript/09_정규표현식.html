졍규표현식

정규표현식(Regular Expression : REGX / REGEXP)이란,
특정 패던을 가진 문자열을 찾거나 변경할 떄 사용하는 형식 언어
String 클래스 => mathes()
정규표현식을 사용하면 조건검사를 하거나 값을 변경할때
조건문 / 반복문을 사용하지 않고 간단히 처리가능
자바스크립트 뿐만 아니라 자바 / 오라클에서도 사용할 수 있다

*메타문자

메타문자를 사용해서 "검색하고자 않는 패턴" 을 생성할 수 있음
