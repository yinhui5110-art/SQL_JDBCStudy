* jQuery사용하기

jQuery를 사용해서 DOM을 다룰 예정
jQuery코드가 문서상 모든 요소들이 전부 다 로드된 후 에 실행할 수 있게끔 만들어 주어야 한다 => window.onload 

   onload : 페이지가 전부 다 로딩되고나면 자동으로 딱 한 번 수행
   해당 HTML무넛의 요소들도 전부 다 로딩되고 연동되어있는 외뷰 문서들도
   전부 다 로딩된 후에 onload속성에 대입한 이벤트 핸들러가 수행되는 특징을 가진다

   -jQuery에서는 해당 내용을 세 가지 구문을 제공하여 해결하기

   1)
   jQuery(document).ready(function(){
      실행하고싶은내용;
   });
    
    2)
    $ : jQuery키워드를 나타낸다
    $(document).ready(function(){
      실행하고싶은내용;
    });

    3)
    $(function(){
      실행해고싶은내용;
    });

    jQuery에서 해당 function은 HTML문서안에있는 DOM요소들이 모두 로딩되면 바로실행
    여러번 기술 가능, 횟수 제한 없음
